Hi Eivar,

> now I have a question....can I mount an AS for deploy a simple test
> application , that implements the Sh interface using the
> javaDiameterPeer??? 

You can use an AS that supports the ISC interface (SIP) and eventually -
this depends on your application - the Sh interface (diameter).
It is irrespective which technology for the diameter interface is used by
the AS.

For testing purposes you can of course use the Java based Diameter-Peer
offered by Open Source IMS for implementation of Sh interface in your AS.
