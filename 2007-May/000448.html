hi,

"Diameter FQDN" is the new filed in the Web Console, used to specify the
Application Server's peer address. This should follow the same rules as
the Origin-Host Scheme rules and these two will be compared in the
permission-authorization process.

The Server Name, is the "sip:" scheme address to which the request will
be forwarded in the case of the ifcs.

- -Adrian
