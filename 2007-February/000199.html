Hi Katty,

The user and the password for full access to the web interface are:
user: "hssAdmin"
pass: "hss"
It is also available the user "hss" with the same password, but this
offers only read only access.
These are defined in tomcat/conf/tomcat-users.xml of the root directory
of FHoSS.
