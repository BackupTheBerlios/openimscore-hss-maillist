Hi,

I have attached patches for supporting "SIP Digest" authentication
scheme (as defined 3GPP TS 29.228 v8.5.0) in openimscore. I have made
necessary changes to HSS and SCSCF and they are attached as two
separate patches.

I have tested my changes using all openimscore components (P,S,I-CSCFs
and HSS) and using sipp as UA. The changes seem to work fine.

Thanks
Pandu

P.S: I had submitted HSS patch alone about a year ago or so, but the
current patch contains some minor refinements to it.
