Hello Krishnaveni,

I am forward the answer to your mail to the HSS mailing list as well. 

There is a permission list that is attached to each application server
that you create. You should enable the specific commands in the GUI. The
error you get, is related to the fact that the AS you are using is
currently not provisioned in the HSS. Provide it, make sure that the AS
has the appropriate permissions set for Sh-Pull and then try again.

Best regards,

Peter
