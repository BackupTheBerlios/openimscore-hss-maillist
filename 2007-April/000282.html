The proxy flag is defined in standard 3GPP TS 29.329, section 6.1.2,
page 8. Ypou may notice the PXY in the headers, means that proxy flag
must be set.



The User-Data-Answer (UDA) command, indicated by the Command-Code field
set to 306 and the 'R' bit cleared in the Command Flags field, is sent
by a server in response to the User-Data-Request command. The
Experimental-Result AVP may contain one of the values defined in section
6.2 or in 3GPP TS 29.229 [6].



Message Format

< User-Data-Answer > ::= < Diameter Header: 306, PXY, 16777217 > <
Session-Id > { Vendor-Specific-Application-Id } [ Result-Code ] [
