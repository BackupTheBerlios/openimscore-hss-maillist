> The S-CSCFs are configured in the icscf database for which we have
> provided an SQL dump (icscf.sql). The selection is performed
> automatically by the I-CSCF. The HSS can only indicate a set of
> mandatory and optional capabilities for the user to be registered
> to the
> I-CSCF and the I-CSCF will make the decision to which S-CSCF to
> forwards
> the REGISTER request. The first S-CSCF to answer and register the user
> will send MAR/SAR messages to the HSS through the Cx Diameter
> interface
> and the HSS will save the name of the S-CSCF in those requests in the
> S-CSCF field that you were talking about.
>
> -Dragos
