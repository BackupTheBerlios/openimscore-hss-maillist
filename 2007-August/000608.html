Hello Peter,

You need to provision trigger points, application servers, initial
filter criteria and then add this to a service profile which you assign
to users. Well, long story - just try the GUI and there you will
actually find a presence server service already provisioned as an
example - triggers on SUBSCRIBE and PUBLISH and sends them to an app server.
