Hi Alessandro,

For making changes into the HSS please use the admin account:
user: hssAdmin
password: hss
The other account hss/hss is for read only access.

If the HSS does not return any authentication vectors, almost sure you
have not the same scheme set on HSS and in scscf.cfg. Please make a
double check into that file.
If is still not working, please send us an ethereal trace with SIP and
diameter messages, to clarify why is not working your registration.

Cheers,
Adrian
