What do you mean with "registered"? An AS need not be registered. It is addressed via routing in cas an iFC point to the AS.

> But I do not really know how to establish a connection. 

You have to set an IFc which point to the AS. More details at http://www.openimscore.org/faq


> Do I have to use the JavaDiameterPeer?

Only in case of some advanced services you might need a connection to the HSS (Sh interface). If Sailfin does not support Sh interface (diameter) you have to find a solution than. Maybe JavaDiameterPeer might help than.
