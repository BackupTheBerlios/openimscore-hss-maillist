Hi,
 
> I'm looking the HSS tables structure, and I would like to know where is
> the information related to the localization of the user ...

This information is not stored in HSS. The HSS only knows which S-CSCF is
responsible for a user (identified via its Private User ID) after
registration. The location information (content of the Path header field) is
kept in S-CSCF only.
