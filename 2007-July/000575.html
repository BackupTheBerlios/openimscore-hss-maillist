Hi,
When multiple Data Reference AVPs are included in a request (UDR or
SNR), both nodes need supporting the Notif-Eff feature so that so that
these multiple Data References are handled properly.
Me and my team are working on the Supported Features AVP and its
integration with the FHoSS source code. We decided starting with the
SNR command. In fact, we have provided you with some information about
our proposed development together with the source files involved. Now
we are waiting for your comments on that regard.
Maybe we could collaborate in the UDR command in relation to the
Supported Features aswell, if you consider that our cooperation is
valuable. Thanks.

FHoSS by adding multiple "Data-Reference" AVPs to the
UDR (e.g. 'S-CSCFName'  and 'IMSUserState') results in
getting only the first requested one in the result.

and should be supported in the coming months.
