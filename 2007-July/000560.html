Hello all,

A few minutes ago the "01_RevampDB " branch was placed in the trunk of
FHoSS.

To update your current HSS to the last revision, please read the
"install.txt", placed in the root directory of the release, more
carefully, and take also into account that the new database was a little
bit changed.
- From this reason your current provisioned users and profiles have to be
re-added in the new database. A script for creating the new database and
another one which adds 2 demo users were added as usual to the release.
More detailed information you will find in the installation guide.
Please notice also, that the new database name ("hss_db") is different,
so it won't overwrite your current data from the old database ("hssdb").

The web site is in process to be updated these days. Until this will be
finished please follow the installation instructions only from the
"install.txt".

For those of you who want still to play with the old trunk version, you
can still find it here:
svn://svn.berlios.de/openimscore/FHoSS/old/3GPP_Release5/

Regards,
Adrian
