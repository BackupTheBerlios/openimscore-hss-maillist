Thanks for the info.

When I started HSS through the web interface "http://localhost:8080", I
got 

************************************************************************
*****
"A user name and password are being requested by
"http://localhost:8080". The site says "open-ims.org".
************************************************************************
*****

I failed to login and JAVA throw SQL exception, saying hss at open-ims does
not have privileges,...

Then I edited the below line at the end of hss_db.sql of HSS software. 

********************************************************************
grant all on hss_db .* to hss at open-ims.test identified by 'hss';
********************************************************************

Then I could resolve the exception problem and I could login to HSS with
the user names both "hss" and "hssAdmin". 

After this change, If I login as 'hss', only the "search" option is
visible. If I login as hssAdmin, additionally "create" option is
available. But, not change or any other options. 

As per your suggestion, need to "modify" the existing profile, for which
I do not have the option visible in HSS.

Any help on this?
