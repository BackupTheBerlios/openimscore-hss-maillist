Hi Subhashini,

Username hss  does not have persmission to add users.
hssAdmin can create users and edit them.

you should insert the users with the default values to the HSS using the
file "userdata.sql"
Once you successfully insert users you can loggin as hssAdmin edit the
profile. Just click on the search button to see all the usernames. Then
click on the user profile name you want to edit. You should loggin as
hssAdmin.
Hope it helps.

Regards
