Hi Jamal,

The problem that you had presented should appear only if you did not
generated the additional sources, through "ant gen". All the package
de.fhg.fokus.zh.data is automatically generated through "ant gen". So,
please take a look if src-gen folder was generated and contains the
classes from this package in it.
If not, try to regenerate them (the same: "ant gen"). You should receive
an error if are not generated all the classes. If this is the case,
please sent to us that error.

Cheers,
Adrian
