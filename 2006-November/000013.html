Hi Sebastian,

Yes, that is what happens, because the AS might work in proxy mode, if 
it chooses to, and as such the route back has to be indicated. If not, 
as you want it to be work in terminating mode, just answer the request.

Setting it as a final destination is not possible. So you will have to 
modify the Application Server to be ISC-compatible.
