Hi Julio,

Which data exactly do you want? I mean: repositoty data, IMPU, IFC,
location...etc?

User Data is the DIAMETER AVP inside a DIAMETER UDA ANSWER which
allocates the data you have required in the correspondiente UDR
(indicated in "data reference" AVP)


So you first must decide which information do you want and put it in the
"data reference AVP" of your UDR, and you must give the correct
permissions in FHoSS Administration Console (see attachment permissions
for UDR and PUR to ask for and save Repository Data information)

I hope it will help
