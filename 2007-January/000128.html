Hi Manjunath,

As it looks, the problem occurs due to a improper JDBC connection
setting (" No connection properties specified - the user must supply
JDBC connections"). Have you changed anything in
"conf/hibernate.properties"? That file is responsible for specifying the
connection to the mysql database. If not, probably your mysql is not
configured properly on your machine and the HSS can't connect to the
database.

Cheers,
Adrian
