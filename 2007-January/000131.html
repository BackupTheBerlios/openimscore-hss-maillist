Hello,

Yes, you are right, the hibernate.properties file is directly in deploy
directory. If you didn't change it, the problem relates to your mysql
configuration.

Please check this issues:
 1. Can you connect directly to mysql from command line?
 2. The two mysql scripts completed successfully?
 3. The rights were assigned correctly regarding the user that connects
to database? (the last line from hssdb.sql)

 I didn't try it with mysql-4.1.11-2, and I do not know if this is your
problem or not, but you should try to update it, if still does not
working. My current version of mysql is 5.0.

Cheers,
Adrian
