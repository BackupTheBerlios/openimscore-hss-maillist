Hi Luca,

I remember about a month ago you were looking for someone working on accounting. Incase you were referring to charging in the IMS, we have released an initial implementation of charging for IPtv services for the IMS. The system supports online and offline charging for the UCT advanced IPtv server. It is easy to install and use.

Note: since the UCT IPtv server was only tested with the UCT IMS client, this charging release has only been tested with these systems. Your contribution to the development is highly welcome.

 

For more information on the release, visit
