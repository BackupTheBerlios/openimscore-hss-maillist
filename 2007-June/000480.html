Hi Richard,

The HSS-Selected options would allow you to (almost) fully control the
authentication type per user from the HSS (this is done by sending an
authentication scheme AVP with the value of "Unknown" in the MAR request
for authentication vectors). Unfortunately I think that this feature is
only available in the new HSS branch where the HSS has a full bitmap of
allowable authentication mechanisms/user as well as default algorithms
if the S-CSCF is set to HSS-Selected. The S-CSCF cfg will still be
useful as the HSS configures only what is allowed or default and not
what will actually be used.

-Dragos
