Hi JW,

1 - For your first problem, try to add all the elements in a container
element; e.g.
<ServiceData>
	<container>
		<Address>Pasteur</Address>
		<PhoneNumber>514 234 5432MPEG</PhoneNumber>
	</container>
</ServiceData>

2 - If you want to delete the repository data, the service data is not
needed any more; do NOT include it in the xml file. I think that the
scheme does not support an empty element for ServiceData thus is why the
exception is thrown.

- -Adrian
